# FoundryPress Installation Guide

## 1. Upload Files
Upload all files to your domain root:

/public_html/

## 2. Run Installer
Visit:
https://yourdomain.com/install.php

## 3. Activate License
Go to:
/hub/license

Enter your license key to activate your installation.

## 4. Access Hub
https://yourdomain.com/hub/

## 5. Create Brand
Use the Hub to create your first brand.

## 6. Add Articles
Edit:
/brands/{slug}/_data/articles.json

## 7. View Site
/articles/{brand}/
/articles/{brand}/{slug}

## Notes

- Ensure your hosting allows file write permissions
- The system will create required directories automatically
