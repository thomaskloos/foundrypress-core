# FoundryPress

A lightweight, portable publishing system for building and managing multi-brand content sites.

---

## What is FoundryPress?

FoundryPress gives you full control over your content and publishing system.

You can:

- Create and manage multiple content brands
- Publish structured articles using JSON
- Render clean, fast article pages
- Run everything on your own hosting

No SaaS. No lock-in. Just your system.

---

## Core Features

- Multi-brand architecture
- Clean URL routing
- JSON-based content system
- Built-in Hub for managing brands
- Theme-ready structure
- Lightweight and fast

---

## Optional Pro Features

- SEO control system (title, meta, OG)
- Additional tools and modules (expanding over time)

Pro features are available as an add-on module.

---

## What You Get

- Core FoundryPress system
- Documentation
- License key (based on your plan)
- Optional Pro module (if purchased)

---

## Quick Start

1. Upload files to your hosting account
2. Visit: /install.php
3. Complete the setup
4. Access your Hub at /hub/
5. Create your first brand
6. Start publishing

---

## Requirements

- PHP 8.0+
- Apache (with mod_rewrite enabled)
